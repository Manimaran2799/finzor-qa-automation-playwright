Test Case ID      : TC_001
Test Case Name    : Customer Creation with Valid Data
Module            : Customer Management
Priority          : High
Test Type         : Functional Testing

Description:
Verify that a Maker user can successfully create a new customer by entering all mandatory details and submitting the customer record.

Preconditions:

1. Application should be accessible.
2. Maker user should have valid login credentials.
3. User should be logged into the application.
4. Customer Creation module should be available.

Test Data:

* First Name : John
* Last Name  : Doe
* Aadhaar No : 123456789012

Steps to Reproduce:

1. Login to the application as Maker.
2. Navigate to Customer Creation module.
3. Enter valid Personal Details.
4. Enter valid Address Details.
5. Enter valid KYC Details.
6. Review all entered information.
7. Click Save and Continue.
8. Submit the customer record.

Expected Result:

1. Customer record should be created successfully.
2. System should generate a unique Customer ID.
3. Success message should be displayed.
4. Customer record should appear in Unauthorized Inbox.

Actual Result:

1. Customer record created successfully.
2. Unique Customer ID generated.
3. Success message displayed.
4. Customer record available in Unauthorized Inbox.

Status:
PASS

Automation Status:
Automated using Playwright
